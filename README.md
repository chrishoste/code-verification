## Features

- [x] SwiftLint
- [x] Custom UIButton with animations
- [x] Verification Code uses UITexdField
- [x] Adapts dynamically to the number of digits
- [ ] Toolbar for `Next/Prev/Done` - Buttons for navigation
- [ ] Better validation 

## Requirements

- iOS 13.0+
- Xcode 11
